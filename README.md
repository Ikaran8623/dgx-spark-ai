## 🚀 What This App Does

dgx-spark-ai helps you run GPT-OSS 120B on a local NVIDIA DGX Spark system. It also gives you a simple way to fine-tune your own models and use Cline CLI without a cloud account.

This project is built for people who want local control. Your work stays on your machine. You do not need to send data to a cloud service just to get started.

## 💻 What You Need

Before you start, make sure you have:

- A Windows PC
- Admin access on your computer
- A stable internet connection for the first download
- Enough free disk space for AI models
- An NVIDIA DGX Spark setup or compatible local AI hardware
- NVIDIA drivers already installed
- At least 16 GB of RAM for basic use
- More memory if you plan to work with large models

If you plan to use GPT-OSS 120B, you should have a high-memory GPU system with enough local storage for model files and cache data.

## 🔧 Common Fixes

### The app does not open

- Run it as an administrator
- Check that the install folder still exists
- Make sure the download finished fully
- Reboot your PC and try again

### The model does not load

- Check that you have enough RAM or GPU memory
- Confirm the model file is in the correct folder
- Make sure the file is not damaged
- Try a smaller model first

### The GPU is not detected

- Check your NVIDIA driver
- Restart the computer
- Make sure Windows Device Manager shows the GPU
- Reinstall the driver if needed

### The app runs slowly

- Close other heavy apps
- Use a smaller model
- Move model files to a faster drive
- Check that your system has enough free memory

## 🪄 Tips For Better Results

- Start with a smaller model before loading a large one
- Keep model files in one folder
- Use a fast SSD for best load times
- Leave extra disk space for cache and logs
- Use plain prompts for best results
- Save your working setup once it runs well
